# PRD: Web Page Builder (WPB)

## 목표
사용자가 AI 프롬프트만으로 혹은 직접 코딩하여 웹페이지를 신속하게 제작하고 실시간으로 확인할 수 있는 프론트엔드 작업 도구 개발.

## 사용자
- 웹사이트 구조를 빠르게 초안으로 잡고 싶은 개발자.
- 간단한 프론트엔드 실력이 있지만 AI의 도움을 받아 생산성을 높이고 싶은 사용자.

## 핵심 기능
1. **AI 생성 모드**: 자연어 프롬프트를 통해 HTML/CSS/JS 코드를 자동 생성하고 파싱하여 에디터에 삽입.
2. **수동 제작 모드**: Monaco Editor를 통한 정밀한 코드 수정 및 직접 작성.
3. **실시간 미리보기**: 작성된 코드를 iframe의 `srcdoc`을 이용해 전처리 없이 즉시 렌더링.
4. **결과물 다운로드**: 완성된 결과물을 index.html, style.css, script.js 파일로 즉시 다운로드.
5. **통합 에디터 환경**: AI 생성물과 수동 편집물이 실시간으로 동기화되는 하이브리드 UX.

## MVP 제외 사항
- React/Vue 등 빌드 도구가 필요한 프레임워크 지원.
- 서버 사이드 렌더링(SSR) 및 DB 연동.
- 호스팅 및 도메인 연결 기능.
